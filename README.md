SETUP
========

Ir até a pasta que contém o anexo, extrair os scripts e entrar na pasta

```
$ unzip pltfilter_setup.zip
$ cd pltfilter_setup/
```

Executar setup script

```
$ chmod +x setup
$ ./setup
```

```
$ pltfilter
```
